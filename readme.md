# Panda
[![Panda V0.1](http://img.youtube.com/vi/5nXnwZyIH2g/0.jpg)](https://www.youtube.com/watch?v=5nXnwZyIH2g "Video Title")
* A simple vanilla wow addon that makes quick work of disenchanting.
* Panda provides an updating window with 1 click disenchant buttons for the equipment in your bags.
* Includes filters for bind on pickup, and an easy to use blacklist.
* Click on the minimap icon, or type /panda to open the window.
* Uses animations from [PizzaSauce](https://codeberg.org/Pizzahawaii/PizzaSauce) by PizzaHawaii

# Installation:
1. Click the Code button to the upper right hand corner and select download or click [here](https://github.com/tubtubs/Panda/archive/refs/heads/master.zip).
2. Unzip the download into your Interface/Addons folder in your WoW directory. Eg: *C:\Games\WoW\Interface\Addons*
3. Rename the folder from *Panda-master* to *Panda*
4. Restart WoW and enable the addon from the character selection screen. Ensure your addon memory cap is set to 0 (no limit) or a high number like 256.
5. If you're playing on a standard vanilla server, you will need the icon patch. [Download](https://files.catbox.moe/i1c0mq.zip), and move Patch-I.mpq to your Data folder. Eg: *C:\Games\WoW\Data*

You can also use the [GitAddonsManager](https://gitlab.com/woblight/GitAddonsManager) to install this addon, but it will not install the icon patch.

* If there are any further issues with installation, ensure that *Panda.toc* is in the root folder. There should be no subdirectories. Eg: *C:\Games\WoW\Interface\Addons\Panda\Panda.toc*
